## 概述

调查员的职业选择及其提供的技能点和信用评级。职业体现调查员如何谋生，同时反映调查员的专业领域，以此决定他会更擅长哪些技能。在游戏里，调查员职业的实际影响比较有限，它只用来决定调查员的起始技能，并帮助其完善背景。

---

## 详细内容

### 职业规则

- 选择职业后，使用职业指定的属性计算本职技能点数
- 将计算结果作为百分点分配给职业中列出的技能（本职技能）
- 本职技能点同样应该分配给信用评级技能
- 并非所有技能都需要被分配技能点，但剩余未分配的技能点将会消失
- 每项技能旁边的括号里都有一个基础值数字，分配的技能点应加在基础值上
- 若要创造一个新职业，本职技能的数量不应超过八个
- 标记为 [洛式] 的职业具有典型的洛夫克拉夫特式风格
- 标记为 [现代] 的职业只适用于现代背景的游戏

### 信用评级与生活水平

| 生活水平 | 信用评级范围 | 说明 |
|----------|-------------|------|
| 赤贫 | CR 0 或更低 | 连贫穷水平都达不到，露宿街头 |
| 贫穷 | CR 1-9 | 只能负担最简陋的小屋和粗茶淡饭 |
| 标准 | CR 10-49 | 过得较为舒适，每日三餐 |
| 小康 | CR 50-89 | 生活奢侈而安逸，大型住所 |
| 富裕 | CR 90-98 | 生活极尽奢华，豪宅或庄园 |
| 豪富 | CR 99 | 钱已失去意义，世界最富有的人之一 |

## 核心规则/要点

- 职业决定调查员的本职技能列表、信用评级范围和本职技能点计算方式
- 本职技能点应分配给职业列出的技能和信用评级，未分配的点数将消失
- 技能点加在技能基础值之上，角色卡上填写的是基础值加上分配的技能点
- 信用评级反映调查员在该职业中的层级，在职业限制范围内可自由分配
- 创造新职业时，本职技能数量不应超过八个
- 趋于平衡的职业选择会使调查员团队拥有出色的技能组合

## 调查员手册补充职业

> 来源：[[../sources/coc-investigator-handbook.md|克苏鲁的呼唤第七版 调查员手册]] 第四章

调查员手册在规则书的基础上大幅扩展了职业列表，每个职业都附有详细的背景说明、推荐关系人（Contacts），以及多个子类型。以下列出规则书中未收录的职业。

### 会计师

- **技能：** 会计，法律，图书馆，聆听，说服，侦查，任意其他两项个人或时代特长
- **信用评级：** 30-70
- **本职技能点：** EDU×4
- **推荐关系人：** 生意伙伴，法律界，金融业界（银行，其他会计师）
- **说明：** 可能在企业工作或作为自由会计师，为个体经营者和企业客户担任顾问。他们是优秀的研究者，既勤奋又关注细节。

### 杂技演员

- **技能：** 攀爬，闪避，跳跃，投掷，侦查，游泳，任意两项其他个人或时代特长
- **信用评级：** 9-20
- **本职技能点：** EDU×2 + DEX×2
- **推荐关系人：** 业余运动员圈，体育专栏作家，马戏团，嘉年华管理者

### 演员（含戏剧演员、电影演员）

- **技能：** 艺术（表演），乔装，格斗，历史，两项社交技能（话术、恐吓、取悦、说服），心理学，任意一项其他个人或时代特长
- **信用评级：** 9-40（戏剧演员）；20-90（电影演员）
- **本职技能点：** EDU×2 + STR 或 DEX×2
- **推荐关系人：** 戏剧产业，报刊艺术批评家，演员公会
- **说明：** 1920年代后期出现了有声电影，不少默片时代的明星难以适应。电影演员的薪酬远高于戏剧演员。

### 动物训练师

- **技能：** 跳跃，聆听，博物学，动物驯养，科学（动物学），潜行，追踪，任意一项其他个人或时代特长
- **信用评级：** 10-40
- **本职技能点：** EDU×2 + APP 或 POW×2
- **推荐关系人：** 动物园，马戏团，赞助人，演员
- **说明：** 可以像对人一样对动物使用「心理学」技能。

### 考古学家（原作向）

- **技能：** 估价，考古，历史，外语，图书馆，侦查，机械维修，导航或科学（任一）
- **信用评级：** 10-40
- **本职技能点：** EDU×4
- **推荐关系人：** 赞助人，博物馆，大学
- **说明：** 在1920年代，成功的考古学家会被当成著名冒险家与探险家，名利双收。

### 古董商

- **技能：** 会计，估价，汽车驾驶，两项社交技能（话术、恐吓、取悦、说服），历史，图书馆，导航
- **信用评级：** 30-50
- **本职技能点：** EDU×4
- **推荐关系人：** 本地的历史学家，其他古董商，可能还包括赝造师

### 建筑师

- **技能：** 会计，艺术/手艺（技术制图），法律，母语，计算机或图书馆，说服，心理学，科学（数学）
- **信用评级：** 30-70

### 赏金猎人

### 管家、男仆、女仆

### 精神病院看护

- **技能：** 闪避，格斗（斗殴），急救，两项社交技能（话术、恐吓、取悦、说服），聆听，心理学，潜行

- **说明：** 允许接触医疗记录、药品或其他医疗用品。

### 酒保

- **技能：** 会计，两项社交技能（话术、恐吓、取悦、说服），格斗（斗殴），聆听，心理学，侦查，任意一项其他个人或时代特长

### 猎人

- **技能：** 射击，聆听或侦查，博物学，导航，外语或生存（任一），科学（生物学或植物学），潜行，追踪
- **信用评级：** 20-50
- **本职技能点：** EDU×2 + DEX×2
- **推荐关系人：** 外国政府官员，狩猎监管人员，前客户（大多很有钱），黑市商人与黑社会，动物园主

### 书商

- **技能：** 艺术（文学），历史，图书馆，外语，母语，心理学，任意一项其他个人或时代特长
- **信用评级：** 9-30
- **本职技能点：** EDU×4
- **推荐关系人：** 出版社，文学评论家，历史学家等

### 事务所侦探

- **技能：** 一项社交技能（话术、恐吓、取悦、说服），格斗（斗殴），射击，法律，图书馆，心理学，潜行，追踪
- **信用评级：** 20-45
- **本职技能点：** EDU×2 + STR 或 DEX×2
- **推荐关系人：** 本地执法机构，客户
- **说明：** 世界上有许多著名的侦探机构，如平克顿和伯恩斯调查局。调查人员负责为客户解决各种异常事件、阻止谋杀事件、寻找失踪人员等。

- **推荐关系人：** 其他精神疾病研究者，医生，有时还有执法机构的侦探

- **赝造者、伪造者：** 会计，估价，艺术/手艺（伪造），历史，图书馆，侦查，妙手，任意一项其他个人或时代特长。CR 20-60。EDU×4
- **走私者：** 射击，聆听，导航，一项社交技能（话术、恐吓、取悦、说服），汽车驾驶或驾驶（飞行器或船），心理学，妙手，侦查。CR 20-60。EDU×2 + APP 或 DEX×2
- **混混：** 攀爬，一项社交技能（话术、恐吓、取悦、说服），格斗，射击，跳跃，妙手，潜行，投掷。CR 3-10。EDU×2 + DEX 或 STR×2
- **女飞贼（古典）：** 艺术/手艺（任意），两项社交技能（话术、恐吓、取悦、说服），格斗（斗殴）或射击（手枪），汽车驾驶，聆听，潜行，任意一项其他个人或时代特长。CR 10-80。EDU×2 + APP×2

### 教团首领

- **技能：** 会计，两项社交技能（话术、恐吓、取悦、说服），神秘学，心理学，侦查，任意其他两项其他个人特长
- **信用评级：** 30-60
- **本职技能点：** EDU×4
- **推荐关系人：** 主要的信徒都是普通人。不过首领的魅力越高，信徒当中有电影明星或者富有的寡妇之类名人的可能性就越大

### 白领工人（含职员/主管、中层/高层管理人员）

- **职员/主管：** 会计，语言，法律，图书馆或计算机，两项社交技能（话术、恐吓、取悦、说服），心理学，任意两项其他个人或时代特长。CR 9-20
- **中层、高层管理人员：** 会计，两项社交技能（话术、恐吓、取悦、说服），格斗（斗殴），法律，聆听，操作重型机械，心理学。CR 20-80
- **本职技能点：** EDU×4
- **推荐关系人：** 其他办公室职员（职员/主管）；旧时大学同学，共济会和其他兄弟会组织，地方和联邦政府，媒体和销售人员（中高层管理人员）

### 服务生

- **技能：** 会计，艺术/手艺（任一），闪避，聆听，两项社交技能（话术、恐吓、取悦、说服），心理学，任意一项其他个人或时代特长
- **信用评级：** 9-20
- **本职技能点：** EDU×2 + APP 或 DEX×2
- **推荐关系人：** 顾客，犯罪组织
- **说明：** 在禁酒令时期，售酒场所的服务员是非法职业。

### 饲养员

- **技能：** 动物驯养，会计，闪避，急救，博物学，医学，科学（药学，动物学）
- **信用评级：** 9-40
- **本职技能点：** EDU×4
- **推荐关系人：** 科学家，环保主义者
- **说明：** 可以对动物使用「医学」技能。

### 牛仔

- **技能：** 闪避，格斗或射击，急救或博物学，跳跃，骑术，生存（任一），投掷，追踪
- **信用评级：** 9-20
- **本职技能点：** EDU×2 + DEX 或 STR×2
- **推荐关系人：** 本地企业家，州农业部门，牛仔比赛主办者，艺人

### 工匠

- **技能：** 会计，艺术/手艺（任二），机械维修，博物学，侦查，任意两项其他个人或时代特长
- **信用评级：** 10-40
- **本职技能点：** EDU×2 + DEX×2
- **推荐关系人：** 本地商人，其他工匠和艺术家
- **说明：** 可能的行当包括：家具、珠宝、钟表、陶艺、锻造、纺织、书法、裁缝、木工、书籍装裱、玩具制造、彩色玻璃吹制等。

### 计算机程序员、计算机工程师（现代）

- **技能：** 计算机，电气维修，电子学、图书馆，科学（数学），侦查，任意两项其他个人或时代特长
- **信用评级：** 10-70
- **本职技能点：** EDU×4

- **信用评级：** 9-30

### 通讯记者（记者子类型）

- **信用评级：** 0-10
