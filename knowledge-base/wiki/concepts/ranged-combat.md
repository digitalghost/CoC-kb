## 概述

远程武器和投掷武器的使用规则

---

## 详细内容

### 射击基本规则

处理枪械攻击时，投掷百分骰，将掷骰结果与攻击者的射击技能进行对比（使用适当的射击技能专攻）。

射击检定无需对抗。其难度等级由射程决定，诸多因素都可能为射击检定带来奖励骰或惩罚骰。失败的射击检定不会给对手造成任何伤害。

### 射击和敏捷次序

扣动扳机所需的时间远少于挥舞刀剑与棍棒，因此在敏捷次序中，枪械准备完毕的角色在选择射击时，其敏捷 (DEX)+50。

### 射程与射击难度等级

射击技能检定的难度总是由射程决定。对比武器列表中的基础射程，射击者与射击目标间的距离每超过基础射程两倍，难度等级就提高一级。

| 难度等级 | 射程范围 |
|---------|---------|
| 常规 | 基础射程内 |
| 困难 | 远射程（到基础射程的两倍为止） |
| 极限 | 超射程（到基础射程的四倍为止） |

使用奖励骰或惩罚骰进行进一步调整。

**注意**：进行超射程射击时，只有极限成功才能成功命中目标，而只有大成功（结果为 01）才能造成贯穿。

### 射击调整

在确定难度等级后，守秘人应将所有会导致明显优势或劣势的其他因素纳入考虑。

#### 奖励骰因素

| 因素 | 说明 |
|------|------|
| 抵近射击 | 目标在射击者敏捷的二十分之一米以内 |
| 瞄准 | 在敏捷次序中，角色只能在自己的回合里声明精确瞄准。射击者将在其下一轮的回合中进行射击。如果在射击前没有进行其它行动，射击者获得一颗奖励骰。如果射击者在瞄准时移动或受到伤害，他将失去瞄准带来的优势。通过瞄准只能获得一颗奖励骰 |
| 大目标 | 射击目标的体格为 4 或更高 |

#### 惩罚骰因素

| 因素 | 说明 |
|------|------|
| 寻找掩护 | 意识到自己即将受到射击的目标可以进行闪避检定。如果成功，射击者获得一颗惩罚骰。选择寻找掩护的角色失去下一次攻击机会 |
| 掩护和遮蔽 | 如果射击目标一半以上的面积被遮蔽，射击检定获得一颗惩罚骰 |
| 高速移动的目标 | 射击目标全速移动时（MOV8 或更高），射击者获得一颗惩罚骰 |
| 小目标 | 射击目标的体格为 -2 或更低 |
| 手枪连射 | 许多手枪能在一轮内射击两次或三次。每次检定都带有一颗惩罚骰 |
| 装填弹药 | 在同一回合内，角色可以为枪支装填一颗子弹并立刻击发，但这会带来一颗惩罚骰 |
| 射击近战中的目标 | 当射击对象正在参与近战时，射击者获得一颗惩罚骰。大失败意味射击击中了同伴 |

### 装填弹药规则

- 为任何手枪、步枪或霰弹枪装填两发子弹都需要一整轮的时间
- 更换弹夹需要一轮时间
- 更换机枪弹链需要两轮时间
- 在同一回合内，角色可以为枪支装填一颗子弹并立刻击发，但这会带来一颗惩罚骰

### 手枪连射

许多手枪能在一轮内射击两次或三次。单次射击让射击者可以更精确地瞄准并稳住武器。多次射击则以牺牲精确度为代价换取射速。当角色在一轮内射击两次或三次时，为每次射击分别进行攻击检定，每次检定都带有一颗惩罚骰。

### 自动武器射击

如果使用自动武器，射击者每个战斗轮可以扣动一次扳机。接下来，自动武器会持续进行射击，直到松开扳机或弹药耗尽（这被称为全自动射击）。

#### 短点射

扣动一次扳机发射 2 或 3 发子弹，被认为是射击精度与节省弹药的最佳组合。短点射视为一组包含 2 或 3 发子弹的弹幕（该数值取决于武器，而非角色的技能值）。

#### 半自动射击

与手枪连射相同，每次射击分别进行检定，每次检定带有一颗惩罚骰。

#### 全自动射击

在掷骰前，玩家需要声明射击的轮数（或子弹量），数值由玩家决定，最大值为枪械弹容量。

一组弹幕中子弹的数量不能超过射击者的冲锋枪或机枪技能值的十分之一（向下取整）：取技能值的十位数字即可。无论技能值高低，全自动射击的每组弹幕中的子弹数不会少于 3 发。

角色可以在一轮内进行全自动射击（发射多组弹幕），直至弹药耗尽，但在第一组弹幕之后，命中概率将持续下降。

#### 全自动射击的攻击检定

- **第一次攻击检定**：使用通常规则进行射击检定（根据射程决定难度等级，再应用其他射击调整）
- **第二次及之后的攻击检定**：每次攻击检定都增加一颗惩罚骰（或移除一颗奖励骰）。如果这使惩罚骰累加到第三颗，仍只使用两颗惩罚骰，但难度等级提高一级

射击者的每发射一组弹幕（或短点射），或转换射击目标，都需要进行新的攻击检定。所有射击目标都应处于合理的射界以内（约 60 度角的扇形范围）。

#### 每次攻击检定的处理方式

- 若攻击检定成功：半数子弹命中目标（向下取整，至少 1 发）——为这些子弹的伤害掷骰。每发子弹造成的伤害都要减去护甲值
- 若攻击检定极限成功：所有子弹均命中目标，半数子弹（向下取整，至少 1 发）造成贯穿。每发子弹造成的伤害都要减去护甲值

### 远程武器和投掷武器

远程投射武器（如弓箭）的处理方式与射击类似（目标可以选择寻找掩护）。

投掷武器可以被闪避，处理方式与近战相同。

面对远程（投射）武器或是投掷武器的攻击时，角色不能选择反击，除非该角色与攻击者间的距离小于其敏捷 (DEX) 的二十分之一米。

依靠使用者肉体力量的投掷或投射武器（包括弓和投石索，但不包括弩），其武器伤害需要加上攻击者伤害加值的一半。

### 瞄准具

使用枪械时，望远式瞄准具能够有效地让基础射程翻倍。瞄准具并不会为抵近射击带来优势。

### 支架

利用双脚架、三脚架、或固定装置等支架支撑枪械能够有效地让基础射程翻倍。当枪械从物理上被固定在某一结构（载具或建筑）上时，使用者获得一颗奖励骰。

### 抵近射击调整

抵近射击带来的奖励骰不适用于较长或笨重的武器，例如步枪、长管霰弹枪和弓，因为这类武器难以在近距离瞄准移动中的目标。为家庭防卫设计的短管霰弹枪，或是被锯短的霰弹枪仍然能获得抵近射击的奖励骰。

### 射击掩护后的目标

在目标被部分遮蔽的情况下，攻击者可以选择尝试射穿掩护命中对手。在这种情况下，射击者获得一颗惩罚骰（由部分遮蔽产生），并使用掩护带来的护甲值。

如果对手躲藏的空间较小，守秘人应该提高一级难度等级；如果对手躲藏的空间较大，则继续提高难度等级，或者根据情况要求只在大成功时才能命中。

### 武器故障

枪械容易受到卡壳或哑火的影响。如果任何攻击检定的结果大于等于所使用的武器的故障值（见表 17：武器列表，第 401-405 页），这次射击不仅没有命中——枪械根本没有开火。

- 如果发生故障的武器是左轮手枪、中折式枪械或栓动步枪，故障将只是哑弹
- 杠杆式枪械的故障则是卡壳。处理枪械卡壳需要花费 1D6 战斗轮，以及成功的机械维修或射击技能检定

如果攻击检定同时也是大失败，守秘人可以决定是枪械发生故障，还是发生其他的后果，例如命中同伴，或者使用者弄伤了自己。

### 火力压制

火力压制的目的是迫使一组射击目标"寻找掩护"，因为寻找掩护的目标无法还击。射击者无需指定某一特定目标，而是声明要火力压制一组目标。

希望寻找掩护的目标检定完毕后，守秘人应当从被火力压制的这组角色中随机确定一个或多个射击目标，选择寻找掩护的角色也可能成为目标。

---

## 核心规则/要点

- 射击检定无需对抗，难度由射程决定（常规/困难/极限）
- 枪械准备完毕时 DEX+50 排入敏捷次序
- 超射程只有极限成功才能命中，只有大成功才能贯穿
- 抵近射击（DEX/20 米内）获得奖励骰
- 手枪连射每次检定带惩罚骰
- 全自动射击：弹幕子弹数 = 技能值/10（最少 3 发），后续弹幕递增惩罚骰
- 武器故障值决定卡壳/哑火，大失败可命中同伴
