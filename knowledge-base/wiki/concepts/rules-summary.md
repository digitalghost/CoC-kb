## 详细内容

### 游戏系统规则摘要

#### 进行技能检定

- 掷骰之前，先声明检定的目标。
- 设定检定的难度等级。
- 孤注一掷：玩家是否有理由进行孤注一掷？守秘人可以预示孤注一掷失败的后果。
- 检定成功后，技能获得成长标记。

#### 难度等级

- **常规难度等级**：对立技能或属性小于 50，或这一任务对胜任者构成挑战。玩家的掷骰结果需要小于等于其技能或属性值以取得成功。
- **困难难度等级**：对立技能或属性大于等于 50，或这一任务对专家构成挑战。玩家的掷骰结果需要小于等于其技能或属性的半值以取得成功。
- **极限难度等级**：对立技能或属性大于等于 90，或这一任务已触及人类所能做到的极限。玩家的掷骰结果需要小于等于其技能或属性的五分之一值以取得成功。

#### 大成功与大失败

- **大成功**：掷出 01。
- **大失败**：掷出 100 总会大失败；若检定成功所需的掷骰结果低于 50，则掷出 96-100 均是大失败。

#### 奖励骰和惩罚骰

- 每获得一颗**奖励骰**：进行检定时，除通常投掷的一对百分骰以外，额外投掷一颗十位骰（投掷三颗骰子：一颗个位和两颗十位）。使用掷骰结果更好（通常是更低）的那颗"十位"骰的值。
- 每获得一颗**惩罚骰**：进行检定时，除通常投掷的一对百分骰以外，额外投掷一颗十位骰（投掷三颗骰子：一颗个位和两颗十位）。使用掷骰结果更差（通常是更高）的那颗"十位"骰的值。

#### 对抗检定

当两位调查员互相对抗时使用。双方分别使用选定的技能或属性进行检定。成功等级更高的一方获胜：

- 大成功胜过极限成功。
- 极限成功胜过困难成功。
- 困难成功胜过常规成功。
- 常规成功胜过失败和大失败。
- 平局时，技能（或属性）值更高的一方获胜。如果仍然平局，则陷入僵局或双方重新掷骰。对抗检定不能孤注一掷。

#### 人体极限和技能检定

调查员所能应对的上限是 100+ 其技能值或属性值。从属性值最低的调查员开始，依次从对手的属性值中减去每个调查员的属性值。如此进行，直到对手的属性值降低至可供调查员进行检定。未被用于减少对手属性值的调查员现在可以进行技能检定，难度等级（极限、困难或是常规）由对手被减少后的新属性值决定。对手的属性不能通过这种方法减少至 0 点或者更低；检定总是必要的。

#### 组合技能检定

某些情况允许或要求使用一项以上的技能。只进行一次掷骰，然后将掷骰结果与每个技能相比较。守秘人会指定是否需要所有技能都成功，还是仅需其中一项技能成功。

#### 调查员幕间成长阶段

- 为每项拥有成长标记的技能投掷 1D100。若掷骰结果大于当前技能值或大于 95，该技能提升 1D10 点。
- 若技能提升至 90% 或更高，增加 2D6 点理智值。
- 检查信用评级并回顾经济状况（见幕间成长阶段：就职与信用评级，第 96 页）。
- 尝试回复理智值（见第七章：理智）。
- 回顾调查员背景故事（见修改调查员背景，第 95 页）。

### 近战规则摘要

1. 建立攻击次序：将所有角色的敏捷 (DEX) 从高到低排序，敏捷最高者最先行动。
2. 每位角色进行一个行动：角色可以在战斗中进行攻击、闪避、反击、逃跑、释放法术等。注意：一些 NPC 和怪物可能在一轮内进行多次行动。
3. 声明行动的意图：造成伤害、闪避或是施展战技？
4. 被攻击者可以反击、闪避或施展自己的战技。
5. 双方进行技能检定。
6. 确定谁在对抗中获胜：
   - a) 被攻击者选择反击：成功等级更高者造成伤害，且自己不受伤害。平局时，攻击者在对抗中获胜。若双方均失败则不造成伤害。
   - b) 被攻击者选择闪避：若攻击者取得更高成功等级，则攻击成功，对闪避者造成伤害。否则攻击被闪避，不造成伤害。平局时，闪避者在对抗中获胜并闪避攻击。若双方均失败则不造成伤害。
   - 若被攻击者既不反击也不闪避，见先发制人（突袭），第 106 页。
7. 为成功的攻击掷骰决定伤害（见表 17：武器列表，人类徒手攻击为 1D3）。若攻击者取得极限成功（并非反击），会造成更大伤害：
   - 若攻击者使用贯穿武器：伤害取最大值（即最大武器伤害加上最大伤害加值），并额外投掷一次武器伤害。
   - 若攻击者使用非贯穿武器（如钝器）：伤害取最大值（即最大武器伤害加上最大伤害加值）。

战斗检定不能孤注一掷。

### 射击规则摘要

处理枪械攻击时，投掷百分骰，将掷骰结果与攻击者的射击技能进行对比（使用适当的射击技能专攻）。射击检定无需对抗。难度等级由射程决定，诸多其它因素会为射击检定带来奖励骰或惩罚骰。失败的射击检定不会造成伤害。

已准备的枪械：射击时 DEX+50。

#### 射击难度等级

- 基础射程内：常规难度。
- 远射程（到基础射程的两倍为止）：困难难度。
- 超射程（到基础射程的四倍为止）：极限难度。

使用奖励骰或惩罚骰进行进一步调整。

注意：进行超射程射击时，只有极限成功才能成功命中目标，而只有大成功（结果为 01）才能造成贯穿。

#### 射击调整

确定难度等级后，守秘人应将所有会导致明显优势或劣势的其他因素纳入考虑。

| 情况 | 调整 |
|------|------|
| 目标寻找掩护（闪避检定成功） | 一颗惩罚骰 |
| 目标被部分遮蔽 | 一颗惩罚骰 |
| 抵近射击 (DEX×1/20 米) | 一颗奖励骰 |
| 瞄准（一轮） | 一颗奖励骰 |
| 目标高速移动 (MOV8+) | 一颗惩罚骰 |
| 目标很小 (体格 -2) | 一颗惩罚骰 |
| 目标很大 (体格 4+) | 一颗奖励骰 |
| 装填一发子弹并立即击发 | 一颗惩罚骰 |
| 手枪在一轮内射击 2+ 次 | 每次射击都带有一颗惩罚骰 |
| 射击近战中的目标 | 一颗惩罚骰 |

瞄准：在敏捷次序中，角色只能在自己的回合里声明精确瞄准。射击者将在其下一轮的回合中进行射击。如果射击者在瞄准时移动或受到伤害，他将失去瞄准带来的优势。

射击近战中的目标：大失败将击中同伴。如果多个同伴都在射击路径上，幸运值最低的同伴将被击中。

装填弹药：每个战斗轮可以为手枪、步枪或霰弹枪装填两发子弹。更换弹夹需要一轮时间。更换机枪弹链需要两轮时间。

武器故障：攻击检定的结果大于等于武器的故障值，表示武器故障而未能击发。

#### 全自动射击和短点射

在掷骰前，玩家需要声明射击的轮数（或子弹量）。全自动射击射出的子弹被划分为若干弹幕，每组弹幕分别进行攻击检定。一组弹幕中子弹的数量不能超过射击者技能值的十分之一（向下取整），且不少于 3 发。射击者的每发射一组弹幕或转换射击目标时，都需要进行新的攻击检定。而短点射只需要进行一次攻击检定（一组弹幕）。

处理每次攻击检定：

- 若攻击检定成功：半数子弹命中目标（向下取整，至少 1 发）。每发子弹造成的伤害都要减去护甲值。
- 若攻击检定极限成功：所有子弹均命中目标，半数子弹（向下取整，至少 1 发）造成贯穿。每发子弹造成的伤害都要减去护甲值。如果难度等级为极限，那么最好结果也只是命中目标，无法造成贯穿。

全自动射击和短点射的攻击检定：

- 第一次攻击检定：根据射程决定难度等级，再应用射击调整。
- 第二次及之后的攻击检定：每次攻击检定增加一颗额外的惩罚骰（或移除一颗奖励骰）。如果这使惩罚骰累加到第三颗，仍只使用两颗惩罚骰，但提高一级难度等级。

在全自动射击中转换射击目标时，一些弹药会被浪费（每米/码 1 发子弹）。

### 战技摘要

如果目标并非单纯地造成伤害（如卸除武器），则使用战技进行处理。

1. 比较施展战技的角色及其目标的体格：若施展战技的角色体格更低，每点差值使其获得 1 颗惩罚骰。若差值为 3 或更多，则战技无法施展。
2. 进行攻击检定：和常规近战攻击相同，使用斗殴或适当的格斗技能专精进行检定。像受到通常的攻击一样，对手应选择闪避或反击。
   - 若目标选择闪避：战技施展者使用格斗技能与闪避者的闪避技能进行对抗检定。如果战技施展者取得比闪避者更高的成功等级，则战技施展成功（若平局，则闪避者成功闪避）。
   - 若目标选择反击：战技施展者使用格斗技能与反击者的格斗技能进行对抗检定。如果反击者取得比战技施展者更高的成功等级，则战技施展失败，且反击者对战技施展者造成伤害（若平局，则战技施展成功）。
   - 目标可以选择施展自己的战技：与反击的处理方式相同，但不造成伤害，改为应用战技的效果。
3. 成功施展战技可以使角色达成一个目标。

### 其他战斗情况

#### 寡不敌众

一旦某一角色在一轮内进行过反击或闪避，之后本轮内所有对他发起的近战攻击可以获得一颗奖励骰。每轮攻击次数超过一次的角色或怪物可以在一轮内进行相同次数的闪避或反击，超过该次数后，对手对他们的攻击才能获得奖励骰。注意，某些神话生物永远不会因为调查员的攻击而陷入寡不敌众。

#### 远程武器和投掷武器

远程投射武器（如弓箭）的处理方式与射击类似（目标可以选择寻找掩护，见第 113 页）。投掷武器可以被闪避，处理方式与近战相同（见攻击目标选择反击，第 103 页）。面对远程（投射）武器或是投掷武器的攻击时，角色不能选择反击，除非该角色与攻击者间的距离小于其敏捷 (DEX) 的二十分之一米。

依靠使用者肉体力量的投掷或投射武器（包括弓和投石索，但不包括弩），其武器伤害需要加上攻击者伤害加值的一半。处理投射武器和投掷武器的攻击时，守秘人应设置难度等级，规则与枪械射击相同（见射程与射击难度等级，第 112 页）。

#### 脱离近战

角色可以在自己的回合花费该回合的行动来逃离近战范围，前提是存在逃离路线，并且自身未被压制。

#### 护甲

护甲减少角色受到的伤害：从伤害中扣除护甲值。护甲无法减少法术攻击、毒素、溺水等方式造成的伤害。

| 护甲范例 | 护甲值 |
|----------|--------|
| 厚重的皮夹克 | 1 点 |
| 一战标准钢盔 | 2 点 |
| 3 厘米厚硬木 | 3 点 |
| 现代美军头盔 | 5 点 |
| 重型凯芙拉防弹背心 | 8 点 |
| 军用防暴服 | 12 点 |
| 4 厘米厚防弹玻璃 | 15 点 |
| 3 厘米厚钢板 | 19 点 |
| 大号沙包 | 20 点 |

## 核心规则/要点

- 技能检定三步：声明目标 → 设定难度 → 掷骰
- 常规/困难/极限难度分别对应技能值/半值/五分之一值
- 大成功=01，大失败=100（或96-100当技能<50）
- 奖励骰取更好的十位骰，惩罚骰取更差的十位骰
- 对抗检定：成功等级高者胜，平局时技能值高者胜
- 战斗检定不可孤注一掷
- 射击难度由射程决定，配合奖励/惩罚骰调整
- 全自动射击按弹幕分组检定，后续检定累加惩罚骰
