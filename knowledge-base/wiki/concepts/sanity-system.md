## 详细内容

### 理智系统的核心概念

刚进入游戏的调查员精神健全，神智清醒。但在调查过程中，他们会遭遇天外的恐怖、异星的怪物，还有来自克苏鲁神话的恐怖宇宙真相——这些都会挑战他们正常的心智和感官，将他们推向理智崩溃的边缘。有时，这种体验会过于强烈，调查员若不幸承受不住，将陷入临时性的、期限不定的，甚至永久性的疯狂。

每位调查员都拥有**理智值**，该值会随着游戏进程有所起伏。理智既能决定调查员能否承受克苏鲁神话带来的恐怖，也可以衡量他们面对怎样的骇人情境能不落荒而逃。失去理智值可能导致疯狂。一旦理智值归零，调查员将陷入永久性疯狂，再也无法恢复，也不能再作为玩家角色使用。

游戏中的理智值模仿自 H.P. 洛夫克拉夫特的小说，文中的主角在遭逢匪夷所思的物理现象、超越时空的诡异怪物时，往往会昏倒甚至发疯。用在游戏中，理智值用于记录调查员心智的适应性，衡量调查员对精神创伤的抵抗力。起始理智值较高的角色能够更容易地对精神创伤事件做出合理化解释，或是抑制恐怖的记忆。而理智值较低的角色精神相对脆弱，更易受到负面情绪的影响。

对调查员的心智来讲，克苏鲁神话的宇宙恐怖是最根本的威胁，但其他恐怖但相对正常的现象和事件也可能引发精神紊乱。

### 最大理智值

调查员的理智值在游戏过程中既可增加也可减少，但不能超过调查员的**最大理智值**。每名调查员都拥有最大理智值，起始值为 99。掌握克苏鲁神话知识必定会降低调查员的最大理智值。

**最大理智值 = 99 - 克苏鲁神话技能当前值**

获得克苏鲁神话技能点数时，玩家要等量降低调查员的最大理智值。

### 理智损失程度对照

| 损失量 | 表现 |
|--------|------|
| 1-2 点 | 不适和不安，有什么东西在侵蚀着精神的角落——难以察觉，但确实存在。 |
| 3-4 点 | 恶心或者不对劲的感觉。可能表现为呕吐、短暂的麻木或颤抖。 |
| 5-9 点 | 惧怕、恐慌、迷乱——足以引起临时性疯狂。可能表现为一次严重的精神冲击，导致无法控制的身体反应（全身剧烈抽搐、强烈反胃等），以及对周围人和事物产生疏离感。 |
| 10-19 点 | 扭曲心智的恐怖——须发变白，暴烈的身体反应（对自己以及他人），身体或精神停止运转，急剧的性格变化（临时性的），语无伦次，产生幻觉，或者行为偏向暴力或彻底的恐惧。 |
| 20 点及以上 | 究极无垠的恐怖——远超人类的理解能力。人的精神会因此崩溃，需要相当长的时间才能恢复清醒，即使有办法恢复，这个人也会发生不可逆转的改变。必定导致不定性疯狂。 |

| 情境 | 理智损失 |
|------|----------|
| 突然发现残破的动物尸骸 | 0/1D2 |
| 突然发现尸体或尸块 | 0/1D3 |
| 看见鲜血汇聚成溪流 | 0/1D4 |
| 发现惨不忍睹的人类尸体 | 1/1D4+1 |
| 醒来时被困在棺材里 | 0/1D6 |
| 目击好友惨死 | 0/1D6 |
| 目击食尸鬼 | 0/1D6 |
| 遇见确信已死之人 | 1/1D6+1 |
| 经受酷刑拷打 | 0/1D10 |
| 目睹尸体爬出坟墓 | 1/1D10 |
| 目睹不计其数的断头从天而降 | 2/2D10+1 |
| 目睹伟大的克苏鲁 | 1D10/1D100 |

### 习惯恐惧

有些时候，持续接触同一种神话生物，效果并不会累积——怪物不再是活生生的噩梦，而不过是调查员前进道路上的阻碍。一旦调查员因为遭遇某种怪物损失的理智值达到这种怪物所能造成的理智损失上限，一段时间内，调查员不会再因为遭遇这种怪物而失去更多理智值。

但是要警惕！调查员永远不会真正习惯这些污秽的异种怪物。一段时间后，也许待到调查员的生活恢复到某种正常状态，恐惧便会卷土重来。玩家应当记录每种神话存在分别使调查员损失了多少理智值。在每个调查员幕间成长阶段，玩家要将以上数值各减少 1 点——时间会抚平一切。

### 复合理智检定（可选规则）

规则规定，无论遭遇一个还是多个怪物，理智损失都保持不变。一种增加理智损失的选项是要求玩家对每只怪物分别投掷理智损失，取掷骰结果中的最大值。在同时遭遇多种不同的怪物时，这种方法也行之有效。

> **示例**：哈维同时遭遇 3 只深潜者和父神达贡，需要进行一次理智检定。哈维的玩家理智检定失败，必须对每只怪物分别投掷理智损失。她为每只深潜者各掷 1D6，结果分别是 2、4、5。然后为达贡掷 1D10，结果是 4。哈维失去的理智值等于掷骰结果中的最大值，即 5 点。

### 神话淬炼

当调查员的克苏鲁神话技能点数提升至超过理智值时，他将迎来转变。这名调查员的宇宙观会经历一场范式转移，他的性格、他如何理解自己在宇宙中的位置都会发生不可逆转的转变。从此以后，所有理智损失都会减半。这项转变是永久的，即使调查员的理智值再次超过其克苏鲁神话知识也不会退回原状。

### 疯狂与克苏鲁神话

非神话刺激导致的疯狂不会带来克苏鲁神话知识。然而，每次因神话原因而蒙受创伤后（比如遭遇神话怪物、阅读神话典籍、受到神话法术的影响），调查员都会了解到更多神话知识，这会反映在克苏鲁神话技能上。当角色首次因神话而陷入疯狂，其克苏鲁神话技能提升 5 点。此后每次由神话导致的疯狂（临时性或不定性），都会使其克苏鲁神话技能提升 1 点。

---

## 核心规则/要点

- 理智值是衡量调查员精神韧性的核心属性，归零即永久性疯狂，角色退出游戏。
- 最大理智值 = 99 - 克苏鲁神话技能，掌握神话知识会降低理智上限。
- 理智损失以"成功损失/失败损失"格式表示（如 0/1D6）。
- 理智检定不适用奖励和惩罚骰（自救除外），幸运值也不能花费在理智检定上。
- 疯狂分为三种：临时性疯狂、不定性疯狂、永久性疯狂。
- 习惯恐惧机制：对同种怪物累计损失达到上限后暂时免疫，但幕间成长阶段上限减少 1 点。
- 神话淬炼：克苏鲁神话技能超过理智值时，所有理智损失减半（永久）。
