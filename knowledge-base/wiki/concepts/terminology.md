## 概述

本文档收录《克苏鲁的呼唤》第七版规则书中使用的全部关键术语，按拼音顺序排列。术语涵盖核心机制、属性、技能、战斗、理智、魔法等各方面，共计约 70 余条。

---

## 详细内容

### 基础术语

| 术语 | 说明 |
|------|------|
| 1D100（百分骰） | 游戏中的大部分检定都需要用到 1D100，即百分骰。技能和属性值都用百分比表示。 |
| 1D4、1D6、1D8、1D10 等 | 标识骰子的数量及面数。 |
| APP（外貌） | Appearance，属性的一种，见属性。 |
| CON（体质） | Constitution，属性的一种，见属性。 |
| DEX（敏捷） | Dexterity，属性的一种，见属性。 |
| EDU（教育） | Education，属性的一种，见属性。 |
| INT（智力） | Intelligence，属性的一种，见属性。 |
| NPC（非玩家角色） | 由守秘人扮演的非玩家角色 (Non-Player Character)。 |
| POW（意志） | Power，属性的一种，见属性。 |
| SIZ（体形） | Size，属性的一种，见属性。 |
| STR（力量） | Strength，属性的一种，见属性。 |
| 属性 | 每位调查员都需要掷骰得到八项属性：力量 (STR)、体质 (CON)、体型 (SIZ)、敏捷 (DEX)、外貌 (APP)、智力 (INT)、意志 (POW) 和教育 (EDU)。理智值、魔法值和生命值等次要属性均由这八项属性衍生。属性决定角色的基本能力。 |

### 检定与成功等级

| 术语 | 说明 |
|------|------|
| 半值 | 投掷 1D100 时，与角色技能或属性值的半值相对比，用于困难难度。 |
| 五分之一值 | 投掷 1D100 时，与角色技能或属性值的五分之一值相对比，用于极限难度。 |
| 成功等级 | 分为失败、常规成功、困难成功与极限成功四种。极限胜过困难，困难胜过常规，常规胜过失败。 |
| 常规成功 | 骰点小于等于技能或属性值即为常规成功。 |
| 困难成功 | 骰点小于等于技能或属性的半值即为困难成功。 |
| 极限成功 | 骰点小于等于技能或属性的五分之一值即为极限成功。 |
| 大成功 | 1D100 的掷骰结果为"01"时，检定始终成功，并且能够实现可能达成的最有益的结果。 |
| 大失败 | 当通过检定所需的掷骰结果小于 50，而 1D100 掷出 96-100，或通过检定所需的掷骰结果大于等于50，而 1D100 掷出 100 时，检定始终失败，且通常会带来可能造成的最糟糕的结果。 |
| 难度等级 | 难度等级分为三种：常规难度、困难难度和极限难度。 |
| 技能检定 | 以 1D100 进行的检定，见常规成功、困难成功与极限成功。 |
| 对抗检定 | 双方各自进行检定，对比成功等级，成功等级更高的一方获胜。 |
| 组合技能检定 | 将一次掷骰结果同时与多个技能相对比。根据情况不同，守秘人可能要求其中一个或所有技能都成功。 |
| 灵感检定 | 玩家发现游戏陷入僵局时，他们可以进行灵感检定，守秘人会将他们引回正轨。 |
| 幸运检定 | 当情况受外部环境主导时，用来确定调查员的命运。 |
| 理智检定 | 掷 1D100，小于等于当前理智值则成功。成功的理智检定可能减少较少的理智值或根本不减少。而失败的理智检定总会损失更多理智值，甚至可能导致疯狂。 |
| 目标 | 玩家进行检定所希望实现的效果。 |

### 骰子机制

| 术语 | 说明 |
|------|------|
| 骰子 (D) | 表示需要掷骰来决定游戏中发生的结果。 |
| 奖励骰 | 由守秘人在检定中添加。进行技能或属性检定时，掷百分骰和额外一颗十位骰。现在掷 3 颗骰子，一颗个位和两颗十位。使用结果更好（较低）的十位骰。 |
| 惩罚骰 | 由守秘人在检定中添加。进行技能或属性检定时，掷百分骰和额外一颗十位骰。现在掷 3 颗骰子，一颗个位和两颗十位。使用结果更差（较高）的十位骰。 |
| 孤注一掷 | 如果玩家能够证明孤注一掷的合理性，则为其提供第二次技能检定的机会。孤注一掷失败会带来非常糟糕的后果。战斗、追逐、理智和幸运检定不能孤注一掷。详见预示。 |
| 预示 | 在孤注一掷之前，守秘人可以揭示检定失败后可能发生的后果——如果孤注一掷检定失败，这可怕的后果将会降临在调查员身上。 |
| 后果 | 孤注一掷失败所产生的负面结果。 |

### 技能

| 术语 | 说明 |
|------|------|
| 技能 | 一项明确的知识、技术或身体能力，特别属于调查员的可用能力。 |
| 基础值 | 未经训练的调查员成功使用一项技能的概率。有些技能无法轻易掌握，因此只有 1% 的基础值。基础值标注在角色卡上每个技能旁边的括号里。 |
| 成长标记 | 成功使用技能的调查员应该勾选那项技能旁边的选框（做标记或打勾）。在调查员幕间成长阶段，为标记过的技能进行成长判定，这些技能将有机会提升 1D10 点。克苏鲁神话和信用评级不能获得成长标记。 |
| 调查员幕间成长阶段 | 玩家更新其调查员的阶段（为提升技能进行掷骰等），通常发生在游戏结束或模组结束时。 |
| 兴趣技能 | 创建调查员时，玩家可以向任何技能投入 INT×2 点技能点。 |

### 战斗

| 术语 | 说明 |
|------|------|
| 近战 | 徒手或使用以肉体力量驱动的武器进行战斗，如出拳或挥剑。 |
| 护甲 | 减少所受的伤害。穿着防弹衣、怪物的天然鳞甲都能使其获得护甲。从伤害中减去护甲值，结果才是实际损失的生命值。 |
| 伤害 | 受到攻击和意外受伤被称为"受到伤害"，即对角色造成伤口或损伤。伤害从生命值中扣除。 |
| 伤害加值 (DB) | 由 STR 与 SIZ 相加衍生，见表 1：伤害加值和体格。 |
| 闪避 | 用于回避近战及远程攻击的技能。 |
| 贯穿 | 如果攻击者在攻击检定中取得极限成功，则表示贯穿武器或子弹恰好击中重要部位，会造成更大的伤害。 |
| 寡不敌众 | 角色在一轮中进行过反击或闪避后，之后所有对其的攻击都带有一颗奖励骰。每轮攻击次数超过 1 次的角色或怪物可以在攻击者获得奖励骰之前反击或闪避相同的次数。 |
| 敏捷次序 | 用于战斗和追逐。角色的行动按照各自的敏捷 (DEX) 值从高到低依次进行。 |
| 战斗轮 / 追逐轮 | 弹性游戏时间单位，用于对照多位角色的移动、武器使用及技能使用。轮中的行动次序按照角色的 DEX 从高到低依次进行。 |
| 战技 | 使用格斗技能施展的战术动作，目的是实现单纯造成伤害以外的效果，如擒拿对手，解除武器，使对手失去平衡等。 |
| 突袭 | 出人意料的攻击，可能会让对手措手不及。 |
| 抵近射击 | 目标在距离射击者敏捷值的二十分之一米范围内时，射击者得到奖励骰。 |
| 寻找掩护 | 察觉自己将被射击的目标可以尝试寻找掩护，其进行闪避检定。 |

### 生命与伤害

| 术语 | 说明 |
|------|------|
| 生命值 (HP) | 等于角色的 CON 与 SIZ 之和除以 10 并向下取整。圈出生命值以指示受伤。 |
| 最大生命值 | 角色完全健康时的生命值。 |
| 轻伤 | 单次受到的伤害小于角色最大生命值的一半。角色不会因为轻伤而死亡。 |
| 重伤 | 如果从单次攻击中受到的伤害大于等于角色最大生命值的一半，角色即受到重伤。重伤的角色生命值归零可能导致其死亡。每周结束时，需要进行恢复检定。 |
| 濒死 | 生命值归零且处于重伤状态的角色即陷入濒死。 |
| 昏迷 | 当角色受到重伤时，若未能通过体质 (CON) 检定，其将陷入昏迷。昏迷的持续时间由守秘人决定。成功的急救或医学检定能够唤醒昏迷的角色。 |
| 死亡 | 受到重伤的角色生命值归零时，若未能接受成功的急救（和医学），该角色将会死亡。此外，如果角色在一次攻击中受到的伤害超过其最大生命值，该角色必然死亡。 |
| 急救 | 一项技能，可以为受伤的角色回复 1 点生命值，或者为濒死角色稳定伤势。详见治疗。 |
| 医学 | 为受伤或患病的角色回复 1D3 点生命值。为濒死的角色进行治疗前，需要先通过急救稳定伤势。详见治疗。 |
| 治疗 | 如果受伤的角色未受重伤，该角色可以每天自动回复 1 点生命值，否则需要进行检定，见重伤。急救和医学有助于恢复伤势。 |

### 理智与疯狂

| 术语 | 说明 |
|------|------|
| 理智 | 每个调查员在游戏开始时都是神志清醒的，但随后的理智损失可能导致其陷入临时性、不定性甚至永久性疯狂。 |
| 理智值 (SAN) | 起始值等于角色的 POW。该数值可能发生变动。最大理智值等于 99 减角色的克苏鲁神话技能点数。理智可能减少或增加，但不会自动回复。 |
| 最大理智值 | 等于 99 减克苏鲁神话技能。 |
| 理智损失 | 经由令人不安的可怕经历（比如遭遇怪物）而损失的理智值的数量，通常需要掷骰决定。 |
| 临时性疯狂 | 在单次理智检定中失去 5 点或更多理智值时，调查员的精神将遭受重大创伤。进行一次理智检定，若通过，调查员会理解自己的遭遇，并陷入疯狂 1D10 小时。 |
| 不定性疯狂 | 当调查员在游戏中的一天内失去 1/5 或更多当前理智值时，他将陷入不定性疯狂。不定性疯狂持续到角色被治愈或者恢复为止。 |
| 永久性疯狂 | 如果当前理智值归零，调查员将陷入彻底的疯狂之中（可能需要在收容设施里度过余生），将其从游戏中移除。 |
| 疯狂发作 | 陷入疯狂时，调查员所经历的第一个疯狂阶段。取决于当时的状况，造成即时或总结症状。 |
| 幻觉 | 陷入疯狂时，调查员可能产生幻觉，因此守秘人可以向玩家提供虚假信息。玩家可以通过验明现实检定来看破幻觉。 |
| 验明现实 | 玩家进行一个理智检定来尝试看穿幻觉。 |

### 魔法与神话

| 术语 | 说明 |
|------|------|
| 魔法值 (MP) | 由 POW 属性衍生。魔法值用于施法，是人类和其它智慧生命所固有的。当魔法值归零时，对魔法值的进一步的消耗都将从生命值中一比一扣除。魔法值每小时回复 1 点。 |
| 克苏鲁 | （在混沌元素语中读作 "kuh-THOOloo"）在 H.P. 洛夫克拉夫特的故事中首次登场，此后屡次在其他作者的作品中出场或被提及。 |
| 克苏鲁神话技能 | 这项技能体现角色关于宇宙诸神、神话存在、怪物和恶魔的知识。 |
| 典籍、神话典籍 | 记载着克苏鲁神话知识的书，通常会使读者获得克苏鲁神话技能点数和法术。 |
| 神话等级 | 衡量神话典籍中记载的克苏鲁神话知识深度。 |
| 精读 | 对神话典籍进行系统性研究。同一本神话典籍可以精读多次，每次所需的时间加倍。 |
| 略读 | 对神话典籍进行粗读。浏览全书，获得等于略读值 (CMI) 的克苏鲁神话技能点数。如果读者是相信者，则会造成理智损失。 |
| 深层魔法、深层禁版本 | 比通常版本更有效的法术变体。 |
| 相信者 | 调查员可能间接认识到克苏鲁神话（如通过阅读神话典籍）而并不真的相信神话。当角色第一次正面接触神话时，扣除等同于其克苏鲁神话技能点数的理智值。 |

### 角色与经济

| 术语 | 说明 |
|------|------|
| 调查员 | 玩家角色。 |
| 背景故事 | 用于定义调查员是何种角色。 |
| 职业 | 调查员的生活来源。 |
| 信用评级 | 决定角色可用的现金及资产的数量。还体现角色能够负担的生活水平。 |
| 现金 | 来自信用评级。调查员可以按照他们的意愿随时花费现金。 |
| 资产 | 来自信用评级。以资产形式存在的财富在变现后才能被花费。 |
| 幸运 | 体现角色在其无法掌控的事件上的运气。 |
| 体格 | 用以衡量物理尺寸、力气和耐久性。还可用于体现生物和无生命体的大体比例。 |
| 移动速度 (MOV) | 角色在一轮中可以移动的距离（单位为米或码）。全速奔跑的角色可以移动五倍于该数值的距离。 |
| 人体极限 | 调查员无法与超过自身 100 点或更高的属性相抗衡。 |

### 游戏流程与叙事

| 术语 | 说明 |
|------|------|
| 模组（故事/冒险） | 专为角色扮演设计的一系列故事，包括一系列事件的经过、角色数据、特殊规则和法术，以及调查员可能会发现的有趣或令人回味的描述。模组可能由守秘人自行设计，亦可使用已经出版的模组集或扩展书中的模组。 |
| 战役 | 同一组调查员参加的一系列相互关联的模组。 |
| 守秘人 | 游戏主持者。守秘人知晓故事背后的秘密，负责描述场景，扮演非玩家角色和怪物。守秘人是游戏规则的最终裁决者，应尽力在游戏中保持公正平等。 |
| 可选规则 | 这类规则作为标准规则的补充或替代，由守秘人决定是否使用。 |
| 显明线索 | 守秘人认为不可或缺的线索，应该轻易将之交给玩家。 |
| 隐秘线索 | 如果玩家未能通过察觉检定，守秘人可以决定保留下来的线索。 |
| 展示材料 | 守秘人交给玩家的文件或其它东西，通常是一些线索。 |
| 觉察检定 | 包括侦查、心理学和聆听技能，有时也包括追踪。 |
| 追逐 | 追击或逃离时，在确定的位置进行的一系列行动。追逐可能在驾驶载具时发生，也可能徒步中发生。 |
| 险境 | 在追逐中，险境出现在两个地点之间，可能会使角色受伤，或是减缓角色或载具的速度，通过险境需要进行技能检定。 |
| 知识 / 教育检定 | 某些情况下，调查员所受训练和教育可为其提供额外见解。 |
