# 衍生属性

> 创建日期: 2026-04-13
> 最后更新: 2026-04-13
> 来源数: 1
> 状态: ✅ 已填充

## 概述

由基础属性推导出的 secondary attributes：生命值(HP)、魔法值(MP)、理智值(SAN)、耐力、伤害加值(DB)、体格、移动速度(MOV)和幸运。

---

## 详细内容

### 理智值 (SAN)

- **计算方式：** 理智值起始值等于意志 (POW)
- **说明：** 理智值是《克苏鲁的呼唤》的核心机制之一，代表调查员的精神稳定性。详见第八章：理智。

### 魔法值 (MP)

- **计算方式：** 魔法值等于意志 (POW) 的五分之一
- **说明：** 与意志不同，魔法值可以在游戏中消耗并回复。详见第九章：魔法。

### 生命值 (HP)

- **计算方式：** 将体质 (CON) 与体型 (SIZ) 相加，然后将和除以 10，向下取整
- **公式：** HP = (CON + SIZ) ÷ 10（向下取整）
- **说明：** 生命值用于计算调查员在游戏中累计承受的伤害，指示其在被疼痛、疲惫或死亡压倒前能行动多久。调查员角色卡为帮助玩家追踪生命值与伤害而设计。

### 伤害加值 (DB) 和体格

- **计算方式：** 将力量 (STR) 与体型 (SIZ) 相加，查阅下表确定伤害加值和体格
- **说明：** 高大、强壮的生物和人类能够造成更多物理伤害。在近战中，需要将伤害加值添加至角色造成的伤害里，无论其使用天生武器还是近战武器。伤害加值不适用于枪械攻击。体格用于战技和追逐，也用来体现角色与物体的大体比例。

| STR + SIZ | 伤害加值 (DB) | 体格 |
|-----------|---------------|------|
| 2-64 | -2 | -2 |
| 65-84 | -1 | -1 |
| 85-124 | 0 | 0 |
| 125-164 | +1D4 | 1 |
| 165-204 | +1D6 | 2 |
| 205-284 | +2D6 | 3 |
| 285-364 | +3D6 | 4 |
| 365-444 | +4D6 | 5 |
| 445-524 | +5D6 | 6 |

> 注：之后 STR+SIZ 每增加 80 点，伤害加值 +1D6，体格 +1。

### 移动速度 (MOV)

- **计算方式：** 根据力量 (STR)、敏捷 (DEX) 和体型 (SIZ) 的关系确定基础值，再根据年龄调整

**基础移动速度：**

| 条件 | MOV |
|------|-----|
| DEX 和 STR 皆小于 SIZ | 7 |
| STR 或 DEX 大于等于 SIZ，或三者相等 | 8 |
| STR 和 DEX 皆大于 SIZ | 9 |

**年龄调整：**

| 年龄 | 调整 |
|------|------|
| 40+ | MOV -1 |
| 50+ | MOV -2 |
| 60+ | MOV -3 |
| 70+ | MOV -4 |
| 80+ | MOV -5 |

> 注：非人类角色不适用上述移动规则。

**说明：** 调查员在一轮中所能移动的最大距离等于其 MOV×5 米（或码）。

### 幸运

- **计算方式：** 投掷 3D6 × 5
- **说明：** 幸运值用于游戏中各种需要运气的场合。详见第五章：游戏系统。15-19 岁的调查员在决定幸运值时掷骰两次，取较高值。

---

## 核心规则/要点

- 理智值 (SAN) 起始等于 POW，魔法值 (MP) 等于 POW 的五分之一
- 生命值 (HP) = (CON + SIZ) ÷ 10，向下取整
- 伤害加值和体格由 STR + SIZ 的总和查表决定
- 移动速度由 STR、DEX、SIZ 三者的关系决定，并受年龄影响
- 伤害加值仅适用于近战，不适用于枪械攻击
- 体格用于战技和追逐判定
- 意志的损失在游戏中通常是永久性的，但魔法值可以消耗并回复
